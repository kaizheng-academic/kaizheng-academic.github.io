Installation

git clone https://github.com/kaizheng-academic/kaizheng-academic.github.io
cd academicpages.github.io
bundle install
bundle exec jekyll serve

Use ./code/scripts/new_paper.sh to scaffold new paper pages and ./code/scripts/build_index.sh to regenerate docs overview/example pages.
